# Poverty Point Free-Net
## Created by [Shannon Ellery Hubbell](https://linktr.ee/shannonhubbell)
An interactive horror story in the form of the recovered files from a dialup BBS in 1990s small town Louisiana. Built with [Twine](https://twinery.org/).

This is a work in progress! Keep coming back! [Go here now for a brief "coming soon" prologue.](https://shannonelleryhubbell.itch.io/poverty-point-free-net)
